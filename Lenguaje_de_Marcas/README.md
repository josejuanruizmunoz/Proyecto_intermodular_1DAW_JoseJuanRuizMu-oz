# Proyecto Web - The Bug Fixer Barbers

## Empresa
Esta web es para una barbería ficticia llamada "The Bug Fixer Barbers". La idea es que es una barbería orientada a programadores e informáticos. Por eso el diseño tiene un tema oscuro y usa fuentes y colores (amarillo y azul oscuro) que recuerdan a los editores de código. 

## Páginas que incluye
El proyecto está compuesto por 4 páginas HTML que están enlazadas por un menú de navegación:
- index.html: Es la página de inicio. Tiene una foto principal, los puntos fuertes del local y un ejemplo de antes y después.
- servicios.html: Muestra una cuadrícula con los diferentes cortes, arreglos de barba y sus precios.
- equipo.html: Una página de presentación con la foto de los 3 barberos de la plantilla.
- contacto.html: Tiene la dirección, los horarios y un formulario funcional para pedir cita.

## Cómo abrir la web
No hace falta instalar ningún servidor local ni nada raro. Para ver la web solo hay que:
1. Tener los archivos .html sueltos en la carpeta principal, el archivo .css dentro de su propia carpeta y las fotos en la carpeta "img".
2. Hacer doble clic en el archivo index.html.
3. Se abrirá directamente en el navegador web y ya se puede navegar por los menús.
